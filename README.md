# 찰떡하우스

COVID-19로 인해 UN-TACT 시대가 열렸습니다. 이러한 시대적 상황을 고려하여 부동산 매물을 webRTC를 사용한 화상 통화 서비스를 통해 집에서도 매물을 확인할 수 있는 서비스를 제공하고 해당 매물 근처에 어떤 편의 시설이 존재하는지 알려주는 서비스를 제공하는 플랫폼입니다.
